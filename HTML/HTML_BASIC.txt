static web and Dynamic web
---------------------------
static web: static web pages are fixed and display the same content for every user. they are created using HTML and CSS, and do not require any server-side processing. for example: a personal blog or a company website that only provides information.
Dynamic web:these websites uses server-side programming languages and databases to generate content in real-time, allowing for user interaction and personalized experiences.

for example: social media platforms, e-commerce websites, and online banking portals.

what is html:-Hyper text Markup languages
--------------
HTML is the standard markup language used to create web pages. It consists of a series of elements and tags that define the structure and content of a web page. 
HTML allows developers to create headings, paragraphs, lists, links, images, and other multimedia elements that make up a web page. 
It is the backbone of web development and is essential for building websites and web applications.

                  Block level element and Inline element
-----------------------------------------------------------------------------------------------------------
      Block level element: Block level elements are HTML elements that start on a new line and take up the full width of their parent container. 
                          Examples of block level elements include <div>, <p>, <h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <ul>, <ol>, and <li>.

      Inline element: Inline elements are HTML elements that do not start on a new line and only take up as much width as necessary. 
                      Examples of inline elements include <span>, <a>,<i>, <img>, <strong>, and <em>.

--------------------------------------------------------------------------------------------------------------------------------------------------
                  Div and span tags
------------------------------------------------------------------------------------------------------------------------------------ 
      Div tag: The <div> tag is a block-level element used to group and organize other HTML elements. It is often used for layout purposes, allowing developers to create sections or containers on a web page. 
      Example: <div class="container">...</div>

      Span tag: The <span> tag is an inline element used to apply styles or group small sections of text within a larger block of content.
       It does not create a new line and is typically used for styling specific parts of text.
      Example: <p>This is a <span class="highlight">highlighted</span> word.</p>

                  Datalist tags
---------------------------------------------------------------------------------------------------------------------------------------------------

      The <datalist> tag is used to provide a list of predefined options for an <input> element. 
      It allows users to select from a list of options while still allowing them to enter their own value if desired.
      Syntax:
            <input list="options">
            <datalist id="options">
                  <option value="Option 1">
                  <option value="Option 2">
                  <option value="Option 3">
            </datalist>
      Attributes:
      list="id" ----> the attribute is used to associate the input element with the datalist element by specifying the id of the datalist
      id="text" ----> the attribute is used to give id to the datalist element
      <option> tags are used inside the <datalist> tag to define the options in the list.

                         Add favicons
---------------------------------------------------------------------------------------------------------------------------------------
      A favicon is a small icon that represents a website and is displayed in the browser's address bar, tabs, and bookmarks. 
      To add a favicon to your website, you can use the following HTML code in the <head> section of your HTML document:
      <link rel="icon" href="path/to/favicon.ico" type="image/x-icon">
      Attributes:
      rel="icon" ----> the attribute is used to specify that the linked file is a favicon
      href="path/to/favicon.ico" ----> the attribute is used to specify the path to the favicon file
      type="image/x-icon" ----> the attribute is used to specify the MIME type of the favicon file
